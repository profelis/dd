    dd

    dd is a 2d-framework for flash-based applications with stage3D support. It is written in Haxe with HxSL (Haxe shader language). It is mainly inspired by nd2d.

    This project is in alpha state currently, so use it at your own risk.

    dd

    dd – это 2d-фреймворк для flash-приложений с поддержкой stage3D, написанный на Haxe с использованием HxSL (шейдерного языка Haxe). Основным источником идей для него послужил nd2d.

    В данный момент проект находится в стадии альфа-версии, так что используйте его на свой страх и риск.
